# DIGIVERA - Landing Page

Advanced, premium landing page for DIGIVERA Digital Identity & Footprint Monitoring Platform.

## Features

✅ **Light & Dark Mode** - Premium theme toggle with localStorage persistence
✅ **Fully Responsive** - Mobile-first design
✅ **Performance Optimized** - Fast loading, lazy sections
✅ **Advanced UI/UX** - Cards, gradients, depth effects
✅ **10 Complete Sections** - Navbar to Footer
✅ **Indian Rupees Pricing** - ₹299/month Pro plan
✅ **SEO Optimized** - Proper meta tags and structure

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM
